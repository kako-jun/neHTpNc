# Circular Tetris Experiments

実験的なテトリスゲーム集。TypeScript + Three.jsで作られた3Dレンダリングの2Dテトリスゲームです。

## 特徴

- **3D空間で作られた2Dゲーム**: Three.jsを使用し、ローポリの3Dブロックをorthographicカメラで2D視点に
- **p5.js風のビジュアルエフェクト**: Post-processing (bloom、glitch) による綺麗なビジュアル
- **4つの実験的ゲームモード**:
  - **Classic (4)**: 通常のテトロミノ（4ブロック）
  - **Triomino (3)**: 3ブロックの小さいピース
  - **Pentomino (5)**: 5ブロックの大きいピース（12種類）
  - **Circular**: 外周から中心に向けてブロックが落ちる円形テトリス

## 技術スタック

- **TypeScript**: 型安全な開発
- **Three.js**: 3D WebGLレンダリング
- **Vite**: 高速な開発サーバーとビルドツール
- **Post-processing**: UnrealBloomPass、GlitchPass

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーを起動（http://localhost:3000）
npm run dev

# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 操作方法

- **← →**: ピースを左右に移動
- **↑**: ピースを回転（円形モードでは右に移動）
- **↓**: ピースを下に移動
- **Space**: ハードドロップ（一気に落とす）

## ゲームモード

### Classic (4ブロック)
通常のテトリスと同じ7種類のテトロミノ（I, O, T, S, Z, J, L）

### Triomino (3ブロック)
3ブロックで構成された小さいピース。より素早いゲームプレイ。

### Pentomino (5ブロック)
5ブロックで構成された大きいピース。12種類の形状で難易度アップ。

### Circular
外周から中心に向けてブロックが落ちる円形テトリス。リングが完成すると消える。

## ビジュアル効果

- **Bloom Effect**: ブロックが光って見える発光エフェクト
- **Glitch Effect**: 軽微なグリッチエフェクト
- **3D回転**: ブロックが常に回転してダイナミックな見た目に
- **エミッシブマテリアル**: ブロック自体が光を放つ
- **ポイントライト**: マゼンタとシアンのライトによる演出
